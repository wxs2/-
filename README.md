# - 闲鱼爬虫闲鱼商品数据爬虫
闲鱼爬虫源码淘宝闲鱼商品爬虫源码，可以爬取闲鱼商品的商品内容，图片，价格，出售人信息等等

本源码的功能：

运行本程序之后可以抓取手机闲鱼App中的商品数据，可以随意按照自定义的关键词抓取自己想要的商品数据。包含商品的详细信息，抓取入库了。可自行筛选符合自己要求的商品数据，方便快捷精准买到性价比高的二手商品。
